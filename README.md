bootstrap-uyghur
================

twitter bootstap for uyghur language

* RTL enabled
* Font Embedded(ttf,eot,woff,svg). Font 'alkatip tor' from UighurSoft.
* Uyghur Input(bedit.js by Muhemmed Abdulla @ Yulghun.com)

ئۇيغۇرچە bootstrap
==================

* ئوڭدىن سولغا يۈنۈلۈش قوشۇلغان
